What's included and why?

Your certificate renews automatically. This profile uses SCEP (Simple Certificate Enrollment Protocol), so your device can request a fresh certificate on its own before the current one expires. You install this once and never think about it again.

Your private key never leaves your device. The certificate is generated on-device and signed directly by CSH's certificate authority, so the server never has access to your credentials.

Three CA certificates are bundled in this profile:

  • CSH Root CA — the root of trust for the entire CSH certificate infrastructure.
  • CSH Wireless CA — lets your device confirm it's connecting to a real CSH access point, not an impostor.
  • CSH Code Signing CA — verifies this profile itself was issued by CSH and hasn't been tampered with.

Manual Configuration

Certificates are generated fresh on each download and are valid for 1 year.

Android
  1. Download and install the CA certificate below.
  2. Download and install the .p12 client certificate.
  3. Go to WiFi settings → Add network → select {{.SSID}} → EAP-TLS → choose your certificate.

Windows
  1. Download and install the CA certificate below (import into Trusted Root Certification Authorities).
  2. Download and install the client certificate (.p12) below (import into Personal certificate store).
  3. Download the WLAN profile XML and import it by running: netsh wlan add profile filename="csh-wifi.xml"

Linux

Add a network block to /etc/wpa_supplicant/wpa_supplicant.conf:

network={
    ssid="{{.SSID}}"
    key_mgmt=WPA-EAP
    eap=TLS
    identity="{{.Username}}"
    ca_cert="/etc/wpa_supplicant/ca.pem"
    private_key="/etc/wpa_supplicant/client.p12"
}
